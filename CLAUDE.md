# AI Pulse

AI 業界のニュースを自動収集するエージェント

## プロジェクト概要

Claude Agent SDK の技術検証を目的として、AI 業界のニュースを毎朝自動収集・分析する単一エージェントシステム。

## ルール

### 出力言語
- すべての出力は日本語で行う
- 記事の概要（summary）も日本語で生成する

### カテゴリ定義

| カテゴリ | 説明 | 例 |
|----------|------|-----|
| Model | LLM、基盤モデル、学習手法に関する発表 | GPT-5 発表、Claude 4 リリース、新しい学習手法 |
| Service | API、製品、サービスに関する発表 | API 価格改定、新機能追加、SDK リリース |
| Other | 上記以外 | 人事異動、パートナーシップ、イベント告知 |

### 重要度定義

| 重要度 | 説明 | 例 |
|--------|------|-----|
| high | 新モデル発表、大型買収、重大発表 | GPT-5 発表、1000億ドル買収 |
| normal | その他すべて | マイナーアップデート、ブログ記事 |

### データフォーマット

記事 ID は `YYYYMMDD-NNN` 形式（例: 20251225-001）

### 重複回避
- 既に収集済みの記事（URL ベース）はスキップする
- Sessions を使用して前回状態を参照する
